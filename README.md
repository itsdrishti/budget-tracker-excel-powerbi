# 💰 Financial Budget Tracker

## Project Overview
Personal Finance Budget Tracker built using 
Excel and Power BI.

## Tools Used
- 📊 Microsoft Excel
- 📈 Power BI

## Excel Skills Covered
- SUMIF, COUNTIF, AVERAGEIF
- IF, Nested IF, AND/OR
- VLOOKUP, HLOOKUP, INDEX MATCH
- Conditional Formatting
- Data Validation
- Pivot Tables & Charts
- Sheet Protection

## Key Insights
- Food has highest spending — ₹19,900
- Total Savings — ₹8,000
- Budget Health — Excellent
- High Spending Alert triggered!

## Dashboard
Interactive Power BI dashboard showing:
- Total Amount KPI Card
- Category wise spending Bar Chart
- Monthly trend Line Chart
- Category distribution Pie Chart
